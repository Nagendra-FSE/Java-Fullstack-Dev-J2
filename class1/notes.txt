1. Introduction to Web

The Web, also known as the World Wide Web (WWW), is a system of interlinked hypertext documents and resources accessed via the Internet. It allows users to view and interact with text, images, videos, and other multimedia content through a web browser.

Key Concepts:

Internet vs. Web: The Internet is the global network of computers, while the Web is a service that operates over the Internet.

Web Browser: Software application used to access and display web content (e.g., Chrome, Firefox).

URL (Uniform Resource Locator): The address used to access resources on the web.

HTTP/HTTPS: Protocols used for transmitting data on the web securely.

2. Internet Based Services

The Internet supports a variety of services that are commonly used for communication, data exchange, entertainment, and business.

2.1 Email

One of the earliest and most widely used Internet services.

Allows users to send and receive text messages and attachments.

Examples: Gmail, Outlook, Yahoo Mail.

2.2 World Wide Web (WWW)

A vast collection of web pages linked together.

Accessible using web browsers.

Includes websites, blogs, e-commerce platforms, news portals, etc.

2.3 File Transfer Protocol (FTP)

Used for transferring files between computers on a network.

Commonly used by web developers and administrators.

2.4 Voice over Internet Protocol (VoIP)

Enables voice communication over the Internet.

Examples: Skype, Zoom, WhatsApp Calls.

2.5 Instant Messaging and Chat

Real-time communication between users.

Examples: WhatsApp, Telegram, Slack, Messenger.

2.6 Online Streaming

Accessing audio and video content on demand.

Examples: YouTube, Netflix, Spotify.

2.7 Cloud Computing

Storing and accessing data and applications over the Internet.

Examples: Google Drive, Dropbox, Microsoft OneDrive.

2.8 E-Commerce

Buying and selling of goods and services online.

Examples: Amazon, Flipkart, eBay.

2.9 Social Media Platforms

Platforms for social networking and sharing content.

Examples: Facebook, Instagram, Twitter, LinkedIn.
